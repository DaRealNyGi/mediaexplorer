# Agent Behavior

## Core Principle

Environment problems are more common than code problems.

Verify environment first.

## Preferred Diagnostic Commands

```bash
pwd
which python
python --version
uv --version
ls
```

## Troubleshooting Philosophy

Never assume:

- Correct interpreter
- Correct project folder
- Correct virtual environment
- Correct dependency installation

Verify first.
